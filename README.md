## 关于

这个项目是MooDL的网页前端，仅提供网页显示服务

## 在本地调试本项目

MooDL使用Racet作为前端，您需要提前准备好环境再开始调试：

启动网页：
```
npm start
```

您可以在“http://0.0.0.0:3000”查看预览


你可以通过以下命令打包：
```
npm build
```
